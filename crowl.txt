Для решения задачи с использованием библиотеки Highcharts, мы можем настроить отображение легенды и диаграммы в зависимости от ваших требований. Вот примеры конфигураций для каждого случая.

---

### 1. Легенда под диаграммой в двух столбцах

```javascript
Highcharts.chart('container', {
    chart: {
        type: 'pie'
    },
    title: {
        text: 'Пример Pie диаграммы'
    },
    plotOptions: {
        pie: {
            allowPointSelect: true,
            cursor: 'pointer',
            dataLabels: {
                enabled: true,
                format: '{point.name}: {point.percentage:.1f}%'
            }
        }
    },
    legend: {
        layout: 'vertical',
        align: 'center',
        verticalAlign: 'bottom',
        itemStyle: {
            width: '50%' // Ширина строки в легенде для двух столбцов
        },
        useHTML: true
    },
    series: [{
        name: 'Доля',
        colorByPoint: true,
        data: [
            { name: 'Сектор 1', y: 40 },
            { name: 'Сектор 2', y: 30 },
            { name: 'Сектор 3', y: 20 },
            { name: 'Сектор 4', y: 10 }
        ]
    }]
});
```

---

### 2. Легенда слева и справа от диаграммы (по 2 элемента)

```javascript
Highcharts.chart('container', {
    chart: {
        type: 'pie'
    },
    title: {
        text: 'Пример Pie диаграммы'
    },
    plotOptions: {
        pie: {
            allowPointSelect: true,
            cursor: 'pointer',
            dataLabels: {
                enabled: true,
                format: '{point.name}: {point.percentage:.1f}%'
            }
        }
    },
    legend: {
        enabled: false // Легенду в стандартном формате отключаем
    },
    series: [{
        name: 'Доля',
        colorByPoint: true,
        data: [
            { name: 'Сектор 1', y: 40 },
            { name: 'Сектор 2', y: 30 },
            { name: 'Сектор 3', y: 20 },
            { name: 'Сектор 4', y: 10 }
        ]
    }]
});

// Добавляем кастомную легенду
document.addEventListener('DOMContentLoaded', function () {
    const container = document.getElementById('container');
    const legendHtml = `
        <div style="display: flex; justify-content: space-between; margin-top: 20px;">
            <div style="text-align: left;">
                <p>Сектор 1</p>
                <p>Сектор 2</p>
            </div>
            <div style="text-align: right;">
                <p>Сектор 3</p>
                <p>Сектор 4</p>
            </div>
        </div>`;
    container.insertAdjacentHTML('beforeend', legendHtml);
});
```

---

### Объяснение:
1. **Легенда под диаграммой:**
   - Устанавливается `layout: 'vertical'` и `align: 'center'`.
   - Используется `itemStyle` для ограничения ширины строк, чтобы легенда автоматически разбивалась на два столбца.

2. **Легенда слева и справа:**
   - Отключаем стандартную легенду (`legend.enabled: false`).
   - Создаем кастомную легенду через HTML/CSS и размещаем ее вокруг диаграммы.

### Подключение
Убедитесь, что Highcharts подключен в вашем проекте:

```html
<script src="https://code.highcharts.com/highcharts.js"></script>
<div id="container" style="width: 100%; height: 400px;"></div>
```

Попробуйте оба варианта и адаптируйте под свои нужды!
