Для реализации первого варианта (легенда из двух столбцов по два элемента) можно использовать свойства `itemWidth` и `layout: 'horizontal'` в конфигурации легенды Highcharts. Это позволит разбить элементы легенды на два столбца и разместить их по центру под диаграммой.

Вот исправленный код:

```html
<script type="text/javascript">
    $(function() {
        Highcharts.chart('container', {
            chart: {
                type: 'variablepie'
            },
            plotOptions: {
                series: {
                    // общие настройки для всех серий
                },
                variablepie: {
                    center: ['50%', '50%'],
                    allowPointSelect: true,
                    cursor: 'pointer',
                    dataLabels: {
                        enabled: true,
                        format: '{point.name}: {point.percentage:.0f}%'
                    },
                    showInLegend: true
                },
            },
            title: {
                text: 'Testing variable-pie chart'
            },
            tooltip: {
                headerFormat: '',
                pointFormat: '<span style="color:{point.color}">\u25CF</span> <b> {point.name}</b><br/>' +
                    'data <b>{point.y}</b><br/>' +
                    'PrintFormat args (z-index) <b>{point.z}</b><br/>'
            },
            series: [{
                minPointSize: 10,
                innerSize: '20%',
                zMin: 0,
                name: 'countries',
                data: [
                    { name: 'Huge category', descr: "description 1", y: 400, z: 4 },
                    { name: 'Big category', descr: "description 2", y: 300, z: 3 },
                    { name: 'Medium category', descr: "description 3", y: 200, z: 2 },
                    { name: 'Small category', descr: "description 4", y: 100, z: 1 },
                ]
            }],
            legend: {
                enabled: true,
                layout: 'horizontal',  // Горизонтальная компоновка
                align: 'center',       // Центрирование
                verticalAlign: 'bottom',  // Легенда под графиком
                itemWidth: 150,         // Ширина одного элемента легенды
                itemStyle: {
                    fontSize: '14px',
                    fontWeight: 'normal',
                },
                useHTML: true
            }
        });
    });
</script>
```

### Основные моменты:
1. **Легенда в два столбца:**
   - Устанавливаем `layout: 'horizontal'` для горизонтального расположения элементов.
   - Указываем `itemWidth`, чтобы контролировать ширину каждого элемента. Это помогает разделить элементы на два столбца.

2. **Выравнивание:**
   - `align: 'center'` — для центрирования.
   - `verticalAlign: 'bottom'` — для размещения легенды под диаграммой.

3. **Кастомизация текста:**
   - В данном случае для каждого элемента легенды используется `name`.
