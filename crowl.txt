Для построения графика типа `variablepie` в Highcharts вам нужно подготовить массив данных, который будет содержать информацию о каждом секторе графика. Каждый элемент массива должен быть объектом с полями `name` (название сектора), `y` (значение сектора, которое будет отображаться как процент от общего объема) и, при необходимости, `color` (цвет сектора).

Вот пример массива данных для построения графика `variablepie`:

```javascript
var datas = [
    { name: 'Категория 1', y: 30, color: '#FF0000' },  // 30% для Категории 1 с красным цветом
    { name: 'Категория 2', y: 50, color: '#00FF00' },  // 50% для Категории 2 с зеленым цветом
    { name: 'Категория 3', y: 20, color: '#0000FF' }   // 20% для Категории 3 с синим цветом
];
```

### Объяснение структуры:
- **`name`**: Название сектора, которое будет отображаться в легенде и (или) на графике.
- **`y`**: Значение сектора в процентах, которое определяет его размер. В `variablepie` эта величина будет влиять на радиус сектора.
- **`color`** (необязательное): Цвет сектора. Если вы не указываете цвет, Highcharts автоматически назначит его.

### Пример использования массива данных в графике:

```javascript
return {
    template: {
        type: 'png',
        options: {
            chart: {
                backgroundColor: 'rgb(246, 248, 255)',
                type: 'variablepie'
            },
            title: null,
            plotOptions: {
                variablepie: {
                    center: ['50%', '50%'],
                    size: '50%',
                    minPointSize: 10,
                    innerSize: '30%',
                    dataLabels: {
                        enabled: false  // Отключаем подписи для секторов
                    }
                }
            },
            series: [{
                type: 'variablepie',
                name: 'Outer Items',
                data: datas,  // Используем массив данных для построения графика
                zMin: 0,
                zMax: 100,
                id: 'outerItems',
                colors: datas.map(item => item.color)  // Применяем цвета из данных
            }],
            legend: {
                enabled: true,  // Включаем легенду
                align: 'left',  // Легенда слева от графика
                verticalAlign: 'middle',  // По вертикали по центру
                layout: 'vertical',  // Вертикальное расположение
                itemStyle: {
                    padding: '10px',  // Отступы в элементах легенды
                    fontWeight: 'normal',  // Нормальный шрифт
                    fontSize: '14px'  // Размер шрифта
                },
                labelFormatter: function() {
                    return this.name;  // Отображение только имени элемента
                }
            },
            responsive: {
                rules: [{
                    condition: {
                        maxWidth: 750
                    },
                    chartOptions: {
                        series: [{
                            id: 'outerItems'
                        }]
                    }
                }]
            },
            exporting: {
                sourceWidth: getField(metaInf, "width", 786),
                sourceHeight: getField(metaInf, "height", 526),
                allowHTML: true,
                type: 'image/png'
            },
            credits: {
                enabled: false
            }
        }
    },
    warning: warning == "" ? "OK" : warning
}
```

### Примечания:
1. В массиве `datas` для каждого сектора указаны:
   - **`name`** — название сектора, которое будет отображаться в легенде.
   - **`y`** — значение сектора (в процентах или числовое значение, которое интерпретируется как размер сектора).
   - **`color`** — цвет сектора. Если его не указать, Highcharts автоматически назначит цвета.

2. График будет отображать сектора с указанными процентными значениями, и цвета будут соответствовать тем, которые указаны в массиве.

3. Легенда будет отображать названия секторов, а их цвета будут связаны с цветами секторов в графике.

Если у вас возникнут дополнительные вопросы по настройке или данным, не стесняйтесь спрашивать!
