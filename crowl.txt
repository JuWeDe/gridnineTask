Можно сделать это более простым способом, используя встроенные методы Java:

String fileName = "ss_ooo_202361.csv";

// Получение расширения файла
String fileExtension = fileName.substring(fileName.lastIndexOf(".") + 1);
System.out.println("Расширение файла: " + fileExtension);

// Получение строки без цифр и расширения
String stringWithoutDigits = fileName.replaceAll("\\d", "").replace("." + fileExtension, "");
System.out.println("Строка без цифр и расширения: " + stringWithoutDigits);

В этом коде:

1. Для получения расширения файла мы используем substring(), начиная с последнего индекса точки в строке fileName.
2. Для получения строки без цифр и расширения мы используем replaceAll() с регулярным выражением "\\d", чтобы удалить все цифры, и затем replace(), чтобы удалить расширение файла.

Вывод этого кода будет таким же, как и в предыдущем примере:

Расширение файла: csv
Строка без цифр и расширения: ss_ooo_

Это более простой и компактный способ достижения того же результата, без необходимости использования циклов или сложной логики.
